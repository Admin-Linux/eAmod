| eAthena:
-----------
        1. main.sql (Obrigatório)

        -----------------------------------------------------------------------------------------------
        | Local: No seu Banco de Dados principal.
        -----------------------------------------------------------------------------------------------
        | Função: Atualiza os arquivos SQL já existentes na sua base de dados atual, ou seja, 
        | atualiza tabelas que foram previamente criadas no banco de dados Main e Logs.
